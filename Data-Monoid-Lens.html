(<>~) :: Monoid c => Setting a b c c -> c -> a -> bSource

Modify the target of a monoidally valued by mappending another value. 

>>> :m + Control.Lens
>>> both <>~ "!!!" $ ("hello","world")
("hello!!!","world!!!")
