    Lenses, Folds, and Traversals

    Project maintained by ekmett Hosted on GitHub Pages — Theme by mattgraham

    Overview

    This package provides families of lenses, isomorphisms, folds, traversals, getters and setters for Haskell.

    Examples

    You can let the library automatically derive lenses for fields of your data types

    import Control.Lens
import Data.Set as Set
import Data.Set.Lens
import Data.Text.Lens

data Flag = Verbose | Help deriving (Eq,Ord,Show) 

data Config a = Config 
  { _flags :: Set Flag
  , _compiler :: a 
  } deriving (Show)

makeLenses ''Config

cfg :: Config String
cfg = Config Set.empty "ghc"

    You can then automatically access them and update them in a purely functional manner:

    ghci> cfg^.flags.contains Verbose
False

    or imperatively:

    toggleVerbosity :: MonadState Config m => m ()
toggleVerbosity = flags.contains Verbose %~ not

    and updates can even change the type of the container:

    compiler %~ view packed :: Config String -> Config Text
