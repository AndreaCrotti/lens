Indexed Folds

type IndexedFold i a c = forall k f. (Indexed i k, Applicative f, Gettable f) => k (c -> f c) (a -> f a)Source

Every IndexedFold is a valid Fold.

Converting to Folds

withIndicesOf :: Functor f => Overloaded (Index i) f a b c d -> LensLike f a b (i, c) (j, d)Source

Transform an indexed fold into a fold of both the indices and the values. 

 withIndices :: IndexedFold i a c             -> Fold a (i, c)
 withIndices :: Simple IndexedLens i a c      -> Getter a (i, c)
 withIndices :: Simple IndexedTraversal i a c -> Fold a (i, c)

All Fold operations are safe, and comply with the laws. However,

Passing this an IndexedTraversal will still allow many Traversal combinators to type check on the result, but the result can only be legally traversed by operations that do not edit the indices. 

 withIndices :: IndexedTraversal i a b c d -> Traversal a b (i, c) (j, d)

Change made to the indices will be discarded.
